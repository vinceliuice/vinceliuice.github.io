先阅读一下

  • Vimix主题

    Vimix是为 GNOME/GTK+ 桌面环境设计的谷歌风格主题，界面十分简洁清新，你可以在Gnome-shell、Unity等多种gtk桌面使用

  • 主题基于

    这个主题基于 nana-4 的 Flat-Plat 主题制作而来，在此感谢 nana-4 的给力作品

    Flat-Plat主题

  • 安装信息

    需要安装的 GTK2 主题引擎：

    1. GTK2 engine Murrine 0.98.1.1 以上版本

    2. GTK2 pixbuf engine 或者 gtk(2)-engines package

  • Fedora/RedHat 系:

    yum install gtk-murrine-engine gtk2-engines

  • Ubuntu/Mint/Debian 系:

    sudo apt-get install gtk2-engines-murrine gtk2-engines-pixbuf

  • ArchLinux:

    pacman -S gtk-engine-murrine gtk-engines

  • 安装或者卸载

    运行 (VimixDark-Gtk-Themes)

    ./VimixDark-installer.sh

    和 (VimixLight-Gtk-Themes)

    ./VimixLight-installer.sh 这两个脚本文件 （或者直接去主页下载主题的安装包进行安装）
